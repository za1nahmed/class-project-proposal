Web Resume

This project is based on Web Resume.

Team Member: Zain Ahmed

Application Overview: This project is a personal resume website created using HTML and CSS based on the standard Lewis University resume format. It includes a web version of the resume and a downloadable, printable PDF version for professional review.

Technolodgy Overview:The resume website is built using HTML for content structure and CSS for styling. Development was done in Visual Studio Code. GitHub is used for version control and code hosting.

Feature List (prioritized list of features that can likely be delivered in the time allowed):This resume website project includes a responsive web page that follows the Lewis University resume format and presents structured content using HTML and CSS. A downloadable PDF version of the resume is linked directly on the page and is professionally formatted for printing. The site includes clean print formatting using CSS to remove unnecessary elements like page numbers when printed. All styling is handled through a separate CSS file for consistency and maintainability. The project is hosted on GitHub for version control and deployed on Azure, making it globally accessible via a secure HTTPS URL. Additional features include basic accessibility through semantic HTML and compatibility across modern web browsers.
